hexo常用指令
hexo g 生成静态文件
hexo s 启动本地服务器进行预览
hexo d 部署到github上发布
hexo clean 清除缓存
hexo new “abc” 新建文章 在source_posts路径下
hexo new page “pageName” 新建页面
